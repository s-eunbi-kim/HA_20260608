# SLAS 요구사항 명세서

문서 ID: SLAS-SRS-001  
문서명: Speed Limit Assist System Software Requirements Specification  
버전: 1.0  
대상: 속도 제한 보조 시스템 시뮬레이터

## 1. 시스템 개요

속도 제한 보조 시스템, SLAS는 전방 카메라 및 내비게이션 정보로부터 제한속도 정보를 수신하고, 운전자에게 제한속도 정보와 속도 상태를 표시하는 운전자 보조 기능이다.

본 실습의 SLAS 시뮬레이터는 실제 차량 제어를 수행하지 않는다. 실습 범위는 입력 신호에 따른 표시 상태, 경고, 안내 출력 판단으로 제한한다.

## 2. 기능 범위

| 포함 범위 | 제외 범위 |
|---|---|
| 제한속도 정보 표시 판단 | 실제 표지판 영상 인식 알고리즘 |
| SLAS 보조 표시 색상 판단 | 실제 클러스터 그래픽 렌더링 |
| 과속 경고 판단 | 실제 제동 또는 가속 제어 |
| 운전자 설정 Offset 적용 | 실제 SCC/MSLA 제어기 연동 |
| 기능 제한 및 입력 오류 표시 | 실제 진단 통신 |
| 표지판 타입 및 도로 컨텍스트 기반 제한속도 선택 | 지도 DB 업데이트 |

## 3. 용어

| 용어 | 의미 |
|---|---|
| SLAS | Speed Limit Assist System |
| 기본 제한속도 정보 | 내비게이션 또는 카메라 정보에 따라 클러스터에 표시되는 제한속도 정보 |
| SLAS 보조 표시 | SLAS 기능이 제한속도 상태를 색상, 경고, 안내로 강조하는 표시 |
| Offset | 제한속도 기준에 더해지는 운전자 설정 속도 보정값 |
| 적용 제한속도 | SLAS가 현재 판단에 사용하는 제한속도 |
| 경고 임계값 | 적용 제한속도와 Offset의 합 |
| 과속 경고 | 차량 속도가 경고 임계값을 초과한 경우 표시되는 경고 |
| 설정속도 변경 안내 | ASSIST 모드에서 현재 ACC/MSLA 설정속도 변경을 운전자에게 안내하는 표시 |
| 전방 카메라 상태 | 전방 카메라가 제한속도 표지판을 인식할 수 있는 상태 |
| 도로 컨텍스트 | 현재 주행 도로가 본선, 출구 도로, 터널, 신규 도로 중 어디에 해당하는지 나타내는 조건 |
| 이전 표시 제한속도 | 이전 실행 Step에서 기본 제한속도 정보에 표시된 제한속도 |

## 4. 운용 모드

| 모드 | 설명 |
|---|---|
| OFF | SLAS 보조 표시, 과속 경고, 설정속도 변경 안내를 출력하지 않는다. 기본 제한속도 정보 표시는 SLAS 모드와 독립적으로 유지될 수 있다. |
| WARNING | 기본 제한속도 정보와 SLAS 보조 표시를 제공하고, 과속 조건에서 경고를 출력한다. |
| ASSIST | WARNING 모드의 표시를 제공하고, 현재 ACC/MSLA 설정속도와 적용 제한속도가 다르면 설정속도 변경 안내를 출력한다. |

## 5. HMI 표시 정책

| 표시 항목 | 값 | 설명 |
|---|---|---|
| 기본 제한속도 정보 | `---`, 30-130 | 제한속도 정보가 없으면 `---`를 표시한다. |
| SLAS 보조 표시 색상 | OFF, GRAY, GREEN, AMBER, RED | SLAS 상태 또는 속도 상태를 색상으로 표현한다. |
| 과속 경고 | ON, OFF | 차량 속도가 경고 임계값을 초과할 때 표시한다. |
| 설정속도 변경 안내 | NONE, UP, DOWN | 설정속도를 올리거나 낮추라는 안내이다. |
| 기능 제한 메시지 | NONE, INITIALIZING, CAMERA_OBSCURED, INVALID_INPUT | 기능 제한 또는 입력 오류 사유를 표시한다. |
| 적용 제한속도 출처 | NONE, CAMERA, NAVIGATION | 적용 제한속도 판단에 사용된 정보 출처를 표시한다. |
| 시스템 상태 | NORMAL, OFF, LIMITED, INITIALIZING, INVALID_INPUT | 시험 결과 확인을 위한 내부 판정 상태이다. |

## 6. 입력

| 논리 입력 | 값 | 설명 |
|---|---|---|
| SLAS 운용 모드 | OFF, WARNING, ASSIST | 운전자 설정 모드 |
| 차량 속도 | 0-250 km/h | 현재 차량 속도 |
| 카메라 인식 제한속도 | 0, 30-130 km/h | 0은 정보 없음을 의미한다. |
| 내비게이션 제한속도 | 0, 30-130 km/h | 0은 정보 없음을 의미한다. |
| 속도 보정값 Offset | -10-+10 km/h | 운전자 설정 속도 보정값 |
| 현재 ACC/MSLA 설정속도 | 0, 30-130 km/h | ACC/MSLA 설정속도. 0은 설정속도 없음 |
| 시동 후 경과 시간 | 0초 이상 | 시동 후 경과 시간 |
| 전방 카메라 상태 | NORMAL, BLOCKED, LOW_CONFIDENCE | 전방 카메라의 인식 가능 상태 |
| 표지판 타입 | SPEED_LIMIT, MIN_SPEED, UNKNOWN | 인식된 표지판의 타입 |
| 도로 컨텍스트 | MAIN_ROAD, EXIT_ROAD, TUNNEL, NEW_ROAD | 현재 주행 도로 조건 |
| 복수 표지판 감지 여부 | true, false | 동시에 여러 표지판이 감지되었는지 여부 |
| 출구도로 제한속도 | 0, 30-130 km/h | 출구 도로 또는 인접 도로에 해당하는 표지판 제한속도 |
| 카메라 초기화 여부 | true, false | 카메라 초기화 중 여부 |
| 이전 표시 제한속도 | 없음, 30-130 km/h | 외부 입력 신호가 아니라 이전 실행 Step에서 기본 제한속도 정보에 표시된 내부 상태 값이다. |

이전 표시 제한속도는 사용자가 직접 입력하지 않는다. 현재 Step 실행 결과가 30-130 km/h의 기본 제한속도 정보이면 다음 Step의 이전 표시 제한속도로 자동 저장된다. 전체 초기화를 수행하면 이전 표시 제한속도는 없음으로 초기화된다.

## 7. 판정 정책

### 7.1 상태 판정 우선순위

여러 조건이 동시에 성립하는 경우 시스템은 다음 우선순위에 따라 상태와 출력을 결정한다.

본 실습 시뮬레이터는 입력 오류와 시스템 가용성 상태를 운전자 SLAS OFF 설정보다 먼저 표시한다. 따라서 SLAS 운용 모드가 OFF이더라도 초기화 중이거나 카메라 차단 상태이면 INITIALIZING 또는 CAMERA_OBSCURED 상태가 우선 표시된다.

| 우선순위 | 조건 | 주요 출력 정책 |
|---|---|---|
| 1 | 입력값이 유효 범위를 벗어남 | 시스템 상태 INVALID_INPUT, 기능 제한 메시지 INVALID_INPUT, 과속 경고 OFF, 설정속도 변경 안내 NONE |
| 2 | 시동 후 15초 미만 또는 카메라 초기화 중 | 시스템 상태 INITIALIZING, 기능 제한 메시지 INITIALIZING, SLAS 보조 표시 GRAY |
| 3 | 전방 카메라 상태가 BLOCKED | 시스템 상태 LIMITED, 기능 제한 메시지 CAMERA_OBSCURED, SLAS 보조 표시 GRAY |
| 4 | SLAS 운용 모드가 OFF | 시스템 상태 OFF, SLAS 보조 표시 OFF, 과속 경고 OFF, 설정속도 변경 안내 NONE |
| 5 | 위 조건에 해당하지 않음 | 시스템 상태 NORMAL, 제한속도 선택 정책 및 경고/색상 정책 적용 |

### 7.2 제한속도 선택 정책

| 조건 | 선택 정책 |
|---|---|
| 전방 카메라 상태가 NORMAL이고 표지판 타입이 SPEED_LIMIT이며 카메라 제한속도가 유효함 | 카메라 제한속도를 우선 사용 |
| 전방 카메라 상태가 LOW_CONFIDENCE이고 내비게이션 제한속도가 유효함 | 내비게이션 제한속도 사용 가능 |
| 표지판 타입이 UNKNOWN이고 내비게이션 제한속도가 유효함 | 내비게이션 제한속도 사용 가능 |
| 유효한 카메라 제한속도와 내비게이션 제한속도를 모두 선택할 수 없음 | 기본 제한속도 정보는 `---` |

## 8. 요구사항

| 요구사항 ID | 요구사항 |
|---|---|
| SLAS-SRS-001 | 시스템은 정상 동작 상태에서 유효한 제한속도 정보를 선택할 수 있으면 기본 제한속도 정보에 해당 제한속도를 표시해야 한다. |
| SLAS-SRS-002 | 시스템은 정상 동작 상태에서 유효한 제한속도 정보를 선택할 수 없으면 기본 제한속도 정보에 `---`를 표시해야 한다. |
| SLAS-SRS-003 | 기능 제한 메시지가 NONE이고 SLAS 운용 모드가 OFF이면 SLAS 보조 표시 색상을 OFF로 출력해야 한다. |
| SLAS-SRS-004 | 기능 제한 메시지가 NONE이고 SLAS 운용 모드가 OFF이면 과속 경고와 설정속도 변경 안내를 출력하지 않아야 한다. |
| SLAS-SRS-005 | SLAS 운용 모드가 WARNING 또는 ASSIST이고 기능 제한 메시지가 NONE인 경우, 차량 속도가 경고 임계값 이하이면 과속 경고를 출력하지 않아야 한다. |
| SLAS-SRS-006 | SLAS 운용 모드가 WARNING 또는 ASSIST이고 기능 제한 메시지가 NONE인 경우, 차량 속도가 경고 임계값을 초과하면 과속 경고를 출력해야 한다. |
| SLAS-SRS-007 | SLAS 운용 모드가 WARNING 또는 ASSIST이고 기능 제한 메시지가 NONE인 경우, 차량 속도가 경고 임계값의 90% 미만이면 SLAS 보조 표시 색상을 GREEN으로 출력해야 한다. |
| SLAS-SRS-008 | SLAS 운용 모드가 WARNING 또는 ASSIST이고 기능 제한 메시지가 NONE인 경우, 차량 속도가 경고 임계값의 90% 이상이고 경고 임계값 이하이면 SLAS 보조 표시 색상을 AMBER로 출력해야 한다. |
| SLAS-SRS-009 | SLAS 운용 모드가 WARNING 또는 ASSIST이고 기능 제한 메시지가 NONE인 경우, 차량 속도가 경고 임계값을 초과하면 SLAS 보조 표시 색상을 RED로 출력해야 한다. |
| SLAS-SRS-010 | 기능 제한 메시지가 NONE이고 SLAS 운용 모드가 ASSIST이며 현재 ACC/MSLA 설정속도가 적용 제한속도보다 낮으면 설정속도 변경 안내를 UP으로 출력해야 한다. |
| SLAS-SRS-011 | 기능 제한 메시지가 NONE이고 SLAS 운용 모드가 ASSIST이며 현재 ACC/MSLA 설정속도가 적용 제한속도보다 높으면 설정속도 변경 안내를 DOWN으로 출력해야 한다. |
| SLAS-SRS-012 | 기능 제한 메시지가 NONE이고 SLAS 운용 모드가 WARNING이면 설정속도 변경 안내를 출력하지 않아야 한다. |
| SLAS-SRS-013 | 차량 속도 입력은 0 km/h 이상 250 km/h 이하의 정수 값이어야 한다. |
| SLAS-SRS-014 | 제한속도 입력은 0 또는 30 km/h 이상 130 km/h 이하의 정수 값이어야 한다. |
| SLAS-SRS-015 | 속도 보정값 Offset은 -10 km/h 이상 +10 km/h 이하의 정수 값이어야 한다. |
| SLAS-SRS-016 | 현재 ACC/MSLA 설정속도 입력은 0 또는 30 km/h 이상 130 km/h 이하의 정수 값이어야 한다. |
| SLAS-SRS-017 | 입력값이 유효 범위를 벗어나면 시스템은 기능 제한 메시지를 INVALID_INPUT으로 출력하고 과속 경고와 설정속도 변경 안내를 출력하지 않아야 한다. |
| SLAS-SRS-018 | 입력값이 유효하고 시동 후 15초 미만인 경우 시스템은 기능 제한 메시지를 INITIALIZING으로 출력하고 SLAS 보조 표시 색상을 GRAY로 출력해야 한다. |
| SLAS-SRS-019 | 전방 카메라 상태가 NORMAL이고 표지판 타입이 SPEED_LIMIT이며 카메라 인식 제한속도가 유효하면 카메라 인식 제한속도를 내비게이션 제한속도보다 우선 적용해야 한다. |
| SLAS-SRS-020 | 전방 카메라 상태가 LOW_CONFIDENCE이고 내비게이션 제한속도가 유효하면 내비게이션 제한속도를 적용 제한속도로 사용할 수 있다. |
| SLAS-SRS-021 | 표지판 타입이 MIN_SPEED이면 해당 표지판을 제한속도로 적용하지 않아야 한다. |
| SLAS-SRS-022 | 표지판 타입이 UNKNOWN이고 내비게이션 제한속도가 유효하면 내비게이션 제한속도를 적용 제한속도로 사용할 수 있다. |
| SLAS-SRS-023 | 복수 표지판이 감지되고 현재 도로 컨텍스트가 MAIN_ROAD이면 출구도로 제한속도보다 현재 주행 도로의 제한속도를 우선 적용해야 한다. |
| SLAS-SRS-024 | 복수 표지판이 감지되고 현재 도로 컨텍스트가 EXIT_ROAD이면 출구도로 제한속도를 적용 제한속도로 사용할 수 있다. |
| SLAS-SRS-025 | 입력값이 유효하고 초기화 상태가 아니며 전방 카메라 상태가 BLOCKED이면 기능 제한 메시지를 CAMERA_OBSCURED로 출력해야 한다. |
| SLAS-SRS-026 | 입력값이 유효하고 초기화 상태가 아니며 전방 카메라 상태가 BLOCKED이면 SLAS 보조 표시 색상을 GRAY로 출력하고 과속 경고와 설정속도 변경 안내를 출력하지 않아야 한다. |
| SLAS-SRS-027 | 입력값이 유효하고 초기화 상태가 아니며 전방 카메라 상태가 BLOCKED이면 이전 표시 제한속도를 현재 Step의 유효한 적용 제한속도로 사용해서는 안 된다. |
| SLAS-SRS-028 | 입력값이 유효하고 카메라 초기화 중이면 기능 제한 메시지를 INITIALIZING으로 출력하고 과속 경고와 설정속도 변경 안내를 출력하지 않아야 한다. |
| SLAS-SRS-029 | 기능 제한 메시지가 NONE이 아닌 경우 SLAS 보조 표시 색상은 GREEN, AMBER, RED로 출력되지 않아야 한다. |
| SLAS-SRS-030 | 시스템은 적용 제한속도의 출처를 CAMERA, NAVIGATION, NONE 중 하나로 식별 가능하게 출력해야 한다. |
| SLAS-SRS-031 | 내비게이션 제한속도를 적용 제한속도로 사용한 경우 적용 제한속도 출처는 NAVIGATION이어야 한다. |
| SLAS-SRS-032 | 카메라 인식 제한속도를 적용 제한속도로 사용한 경우 적용 제한속도 출처는 CAMERA이어야 한다. |

90% 기준값은 소수점 반올림 없이 계산한다. 예를 들어 경고 임계값이 85 km/h이면 90% 기준값은 76.5 km/h로 판단한다.

| 조건 | 기본값 |
|---|---|
